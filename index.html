PENGANTAR

Halo dan terima kasih telah menggunakan Template Halaman Arahan HTML Bisnis Aria. Aria adalah template halaman arahan HTML yang dibangun dengan Bootstrap yang menampilkan morphtext, slider teks, pemfilteran grid, detail lightbox, nomor hitungan, permintaan panggilan dan formulir kontak.

Dibangun dengan Bootstrap 4, HTML5, CSS3, PHP dan JavaScript, ini akan membantu Anda mencapai tujuan pemasaran online Anda dengan cara yang indah dan efisien..

Kerangka kerja dan toolkit yang digunakan: Bootstrap 4, Popper, jQuery, jQuery Easing, Validator, Isotope, Magnific Popup, Swiper, CountTo

Anda dapat mengedit file Html, Css, Php, Js template dengan editor teks dan kode pilihan Anda: Notepad ++, Visual Code, : Notepad++, Visual Code, Sublime or any other...

untuk pengeditan gambar anda bisa menggunakan Adobe Photoshopatau editor grafis pilihan Anda.

STUKTUR FILE

Documentation - adalah folder dari file bantuan ini.
Web - contains the html, css, js and php files of the template.

Web folder includes index.html file yang akan Anda gunakan untuk membuka template. Selain index.html, Anda juga akan menemukan folder berikut:

  • Css - Bootstrap, Font Awesome, Magnific Popup, Swiper and the template's custom stylesheet styles.css
  • Images - Images used throughout the template including the favicon and logo
  • Js - JavaScript files for Bootstrap, Popper, jQuery, jQuery Easing, Magnific Popup, Swiper, Validator and Aria's custom scripts: scripts.js
  • Php - PHP script files for the contact forms
  • Webfonts - Font Awesome files

mengubah logo

- Tempatkan file gambar Anda di gambar folder
- Terbuka untuk pengeditanindex.html
- temukan Navbar bagian
- Di sini temukan baris kode ini: 

<a class="navbar-brand logo-gambar" href="index.html"><img src="gambar/logo.svg" alt="alternative"></a>

- dan ganti logo.svg dengan nama file baru Anda
- Simpan file dan segarkan jendela browser untuk melihat perubahannya

Mengubah Gambar Latar Belakang Header

- Edit dan simpen gambar ini ke file gambar folder
- buka untuk edit css/styles.css
- temukan Header bagian dan baris kode berikut: 

.header {
	latar belakang: linear-gradient(rgba(0, 0, 0, 0.3), rgba(0, 0, 0, 0.5)), url('../images/header-latar belakang.jpg') center center no-repeat;
	latar belakang-size: cover;
}

- dan ganti latar belakang.jpg dengan nama file Anda sendiri
- Simpan file dan segarkan jendela browser untuk melihat perubahannya

Mengubah Gambar Latar Belakang Bagian Detail

- Edit dan simpen gambar ini ke file gambar folder
- buka untuk edit css/styles.css
- temukan bagian Details 1
- Di sini fokus pada baris berikut: 

.accordion .area-1 {
	height: 27rem;
	background: url('../gambar/details-1-latar belakang.jpg') center center no-repeat;
	background-size: cover;
}

- dan ganti details-1-background.jpg dengan nama file Anda sendiri
- Untuk hasil optimal dengan jumlah pekerjaan paling sedikit, saya sarankan menggunakan ukuran gambar yang sama seperti aslinya di Pratinjau Langsung
- Simpan file dan segarkan jendela browser untuk melihat perubahannya
- Hal yang sama berlaku untuk Detail nya 2 bagian
- Hanya kali ini Anda fokus pada baris berikut 
.tabs .area-2 {

IKON

Ikon Font Keren

Aria menggunakan Font Awesome ikon font di banyak bagian dan elemen.

Untuk mengubah salah satu ikon Font Awesome, Anda cukup mengganti kelas yang akan Anda temukan di index.html mis. fa-binoculars, fa-list-alt, fa-chart-pie dengan yang Anda butuhkan http://fontawesome.io/icons/

Social Media Icons

Untuk mengubah ikon media sosial di bagian Footer template, Anda perlu:
- bukaindex.html dari folder template
- Temukan bagian Media Sosial di Footer:
- dan ganti fa-facebook, fa-twitter,dan seterusnya ... dengan nama kelas yang Anda butuhkanhttp://fontawesome.io/icons/ in the Brand Icons section
- Juga jangan lupa gantihref="#your-link" dengan tautan ke halaman sosial atau akun Anda

tip media sosial

Jika Anda ingin mengontrol tampilan posting ketika Anda membagikan halaman arahan di media sosial (Facebook, Google+ dan LinkedIn khususnya) jangan lupa untuk memperbarui OG Meta Tags (OpenGraph) di bagian Kepala dariindex.html file. 

<meta property="og:site_name" content="" /> <!-- nama wibsite -->
<meta property="og:site" content="" /> <!-- link website -->
<meta property="og:title" content=""/> <!-- judul yang ditampilkan di postingan bersama yang sebenarnya -->
<meta property="og:description" content="" /> <!-- deskripsi yang ditampilkan di postingan bersama yang sebenarnya -->
<meta property="og:image" content="" /> <!-- link gambar, pastikan itu jpg -->
<meta property="og:url" content="" /> <!-- ke mana Anda ingin postingan Anda ditautkan -->
<meta property="og:type" content="article" />

FORMULIR KONTAK

Untuk menerima email dengan data dari pengunjung Anda cukup mengubah baris $EmailTo = "yourname@domain.com"; ke alamat email Anda di php folder di file berikut:

  • callmeform-process.php
  • contactform-process.php
  • privacy-process.php

perhatian:Skrip hanya berjalan online (di server hati dengan dukungan PHP) sehingga formulir tidak akan berfungsi saat halaman arahan offline di hard drive Anda

Jika Anda ingin mengubah nama kolom input, Anda perlu memperbaruinya sesuai dengan file berikut:

  • index.html, scripts.js dan file php dari php folder.

Siapkan Halaman Pengalihan Pada Pengiriman Formulir

Untuk menyiapkan pengalihan ke halaman terima kasih pada pengiriman formulir, Anda perlu:
- buka js/scripts.js di dalam folder template
- Temukan blok kode berikut di bagian formulir 

function cformSuccess() {
	$("#ContactForm")[0].reset();
	csubmitMSG(true, "Message Submitted!")
}

- And change it to: 
function cformSuccess() {
	$("#ContactForm")[0].reset();
	window.location.href = "https://yourlinkhere.com";
}

- Dan ganti di dalam tanda kutip https://yourlinkhere.com dengan tautan anda

Menambahkan Proyek Baru

Untuk menambahkan proyek baru di bagian proyek, Anda perlu:
- buka untuk mengedit index.html dan temukan Projects bagian
- Di sini fokus pada area kode tepat di bawah 

<div class="grid">

- Anda dapat menyalin bagian kode berikut: 
<div class="element-item development">
	<a class="popup-with-move-anim" href="#project-1"><div class="element-item-overlay"><span>Online Banking</span></div><img src="images/project-1.jpg" alt="alternative"></a>
</div>

- Dan tempel di mana pun Anda ingin proyek Anda berada dalam urutan Tampilkan Semua
- Sekarang ganti nama dan gambar sesuai kebutuhan Anda
- Tambahkan juga ke kategori yang ditentukan. Sekarang sudah siap development seperti yang Anda lihat dari kode tetapi Anda dapat mengubahnya ke kategori apa pun yang Anda tentukan

Menambahkan Kategori Baru

Untuk menambahkan kategori proyek baru di bagian proyek, Anda perlu:
- buka untuk mnegedit index.html dan temukanProjects bagian
- di sini Filter bagian duplikat baris seperti di bawah ini: 

<a class="button" data-filter=".development"><span>DEVELOPMENT</span></a>

- Dan ganti kata "pengembangan" di kedua lokasi dengan nama kategori Anda
- Kemudian tambahkan juga ke baris proyek di bawah ini 
<div class="element-item development">

-Dengan mengganti "pengembangan" dengan nama kategori Anda
